# ANZ home loan opt
Optimize financial cost under ANZ home loan policy

![](https://shields.io/badge/dependencies-Julia_1.15-purple)

## Usage

Run the following command in terminal. (Exempt if it's executed within one week ago.)

```bash
julia get_data/mortgage_rate.jl
julia get_data/ocr.jl
julia get_data/wholesale_swap_rate.jl
```
